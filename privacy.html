# Privacy Policy for Football Stadium Weather Chrome Extension Last updated: [Dec. 5, 2024] ## Overview Football Stadium Weather is a Chrome extension that provides weather information for football stadiums. We are committed to protecting your privacy and being transparent about our data practices. ## Data Collection Our extension collects and stores the following information locally on your device: - User preferences (temperature units and dark mode settings) - Temporary weather data cache for offline functionality - No personal information is collected or stored ## Data Usage - Weather data is fetched from OpenWeather API through our secure proxy server - Location data (stadium coordinates) is pre-stored and not collected from users - All data processing occurs locally within your browser ## Data Storage - All data is stored locally using Chrome's storage API - No data is transmitted to external servers except for weather API requests - Cache data is automatically cleared when you clear your browser data ## Third-Party Services We use: - OpenWeather API for weather data - Vercel for API proxy services ## Updates This privacy policy may be updated occasionally. Users will be notified of any significant changes through the extension's update notes. ## Contact For any questions regarding this privacy policy, please contact [Your Contact Information]
